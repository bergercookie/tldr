# ranger commands

> A VIM-inspired filemanager for the console - Cheatsheet.

- Open a file with a specific application:

`:open_with {{application_name}}`

- Open a file with a specific application - in the background/fork:

`:open_with -f {{application_name}}`

- Open a file with a specific application - as root:

`:open_with -r {{application_name}}`

- Open current file only - disregard selected files:

`:open_with -c {{application_name}}`

- Cut all files/directories until end

`dG`
