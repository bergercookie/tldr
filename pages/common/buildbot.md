# buildbot

> Buildbot is a software development continuous integration tool which automates
> the compile or test cycle required to validate changes to the project code
> base.
> More information: <https://buildbot.net/>.

- Reload buildbot configuration (doesn't handle new python files, doesn't restart):

`buildbot reconfig {{path/to/master/dir}}`

- Do complete buildbot restart

`buildbot restart
